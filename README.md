# State of the Map 2019 Website

## Local installation

### Install Jekyll

See http://jekyllrb.com/docs/installation/

### View locally

* `git clone git@github.com:openstreetmap/stateofthemap-2019.git`
* `cd stateofthemap-2019`
* `jekyll serve -wl`
* Point your browser to `http://localhost:4000/`
