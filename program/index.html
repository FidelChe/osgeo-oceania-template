State of the Map

State of the Map 2019 is taking place from September 21st to 23rd 2019 in Heidelberg. The program will allow attendants to present and learn about latest developments in OpenStreetMap in presentations, lightning talks as well as workshops and birds of a feather sessions. Additionally, for the second time, State of the Map will host a parallel Academic Track to showcase scientific contributions on OSM.
